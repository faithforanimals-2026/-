# 免費上線部署步驟

## 你會需要兩個免費帳號

- Supabase：資料庫
- Render：網站主機

## 1. 建立 Supabase 資料庫

1. 到 Supabase 建立新專案。
2. 進入 Project Settings / Database。
3. 複製 PostgreSQL 連線字串。
4. 連線字串會類似：

```text
postgresql://postgres.xxxxx:密碼@aws-xxxx.pooler.supabase.com:6543/postgres
```

## 2. 建立 Render 網站

1. 到 Render 建立新的 Web Service。
2. 上傳或連接這個 `online` 資料夾。
3. 設定：

```text
Build Command: pip install -r requirements.txt
Start Command: gunicorn online_app:app
```

4. 新增環境變數：

```text
APP_USERNAME=admin
APP_PASSWORD=你想設定的密碼
DATABASE_URL=Supabase 的 PostgreSQL 連線字串
```

## 3. 第一次開啟

第一次打開 Render 給你的網址時，系統會自動：

- 建立資料表
- 匯入 `seed_products.csv` 裡的商品清單
- 要求輸入帳號密碼

## 4. 日常使用

- 把 Render 網址給工作人員。
- 大家用同一組帳號密碼登入。
- 活動結束後，到「每日結算」匯出 CSV 備份。

## 免費版注意

免費主機可能休眠，第一次打開會比較慢。免費資料庫也可能因閒置被暫停，所以重要活動結束後務必匯出 CSV。
